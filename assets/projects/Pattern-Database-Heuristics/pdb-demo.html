Algorithm Overview

Pattern Abstraction
Create abstract planning tasks by selecting subsets of variables. Reduces state space complexity while maintaining admissibility in heuristic estimates.
h* Computation
Regression search from goal states using Dijkstra's algorithm. Computes minimum distances for all abstract states in O(|S| log |S|) time.
Pattern DB Heuristic
Looks up abstract state in precomputed h* table. Fast evaluation at planning time with guaranteed admissibility and consistency.
Cliques Heuristic
Sum of compatible pattern databases from maximal cliques in compatibility graph. Produces stronger admissible bounds than single patterns.
Variable Independence
Exploits variable independence using clique decomposition. Independent patterns can be safely summed while maintaining admissibility.
Admissibility
Both PDB and cliques heuristics are admissible—never overestimate true distance. Guarantees optimal solutions in A* search.

This simulator demonstrates how pattern databases and cliques heuristics estimate distances in classical planning tasks.

These techniques are fundamental to efficient AI planning and are used in state-of-the-art planners like FastDownward.
