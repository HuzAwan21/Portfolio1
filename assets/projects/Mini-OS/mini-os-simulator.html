Memory Layout
0x00000 Temporary Registers & Data (256 bytes)
0x000FF Process Meta Information (64 bytes)
0x00C8 Status Flags & Process States (8 bytes)
0x03E8 mtime Cache (4 bytes)
0x03E9 Process 0 (Startup) Registers (120 bytes)
0x07D0 Process 1 Registers (120 bytes)
0x0BB8 Process 2+ Registers (120 bytes each)
Exception Handling Flow
1. Exception/Interrupt occurs in user mode
2. Jump to exception_handler (mtvec)
3. Save critical registers to memory
4. Read mcause to determine exception type
5. Dispatch to appropriate handler
6. Execute OS code or system call
7. Restore registers and return to user
