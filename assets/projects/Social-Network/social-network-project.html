📱 Project Overview

The Social Network Platform is a comprehensive Django web application featuring user profiles, content creation, community engagement, and expertise-based content classification. The system supports user relationships (follow/block), content organization with expertise tagging, truth ratings for community-driven fact-checking, and administrative moderation tools.

Core Features:

  • User Profiles - Account management with follow relationships and community memberships
  • Timeline - Real-time feed with posts, citations, and replies
  • Expertise Areas - Automatic content classification based on user expertise
  • Community Features - User communities with moderation and member management
  • Truth Rating System - Community-driven fact-checking and content validation
  • Admin Dashboard - Moderation tools and content management interface

🗄️ Database Models

Core Django models representing the application's data structure:

SocialNetworkUser
• user_id: Integer (Primary Key)
• username: String
• follows: Many-to-Many (Self-referential)
• communities: Many-to-Many
• created_at: DateTime
Post
• post_id: Integer (Primary Key)
• author: ForeignKey (SocialNetworkUser)
• content: Text
• expertise_area: String (Classification)
• created_at: DateTime
• citations: Text (References)
Community
• community_id: Integer (Primary Key)
• name: String
• description: Text
• members: Many-to-Many (SocialNetworkUser)
• moderators: Many-to-Many (SocialNetworkUser)
TruthRating
• rating_id: Integer (Primary Key)
• post: ForeignKey (Post)
• rater: ForeignKey (SocialNetworkUser)
• rating: Integer (1-5 scale)
• created_at: DateTime
🚀 How to Run Locally
Prerequisites
  • Python 3.10+
  • pipenv (Python virtual environment manager)
  • Git
1. Clone & Setup
git clone <repository-url>
cd assets/projects/bdeProject
pipenv install
2. Activate Virtual Environment
pipenv shell
3. Initialize Database (if needed)
python manage.py migrate
python manage.py create_fake_data
4. Run Development Server
python manage.py runserver

Server will start at http://127.0.0.1:8000/

5. Access the Application
  • Homepage: http://127.0.0.1:8000/ - Login interface
  • Timeline: http://127.0.0.1:8000/sn/html/timeline - Main feed
  • Communities: http://127.0.0.1:8000/sn/html/communities
  • Admin: http://127.0.0.1:8000/admin/ - Admin panel
🔌 REST API Endpoints

The application provides comprehensive REST APIs for frontend and external integrations:

GET /api/users/ - List all users
POST /api/users/ - Create new user
GET /api/posts/ - List all posts
POST /api/posts/ - Create new post
GET /api/communities/ - List communities
POST /api/truth-ratings/ - Submit truth rating
