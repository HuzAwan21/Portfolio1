Social Network

Django-based Social Platform with Community Features

🚀 Getting Started

This is a full Django web application with a complete frontend implementation. Follow the steps below to run the platform locally.

Installation & Setup
1. Navigate to the project directory:
cd assets/projects/Social-Network
2. Install dependencies with pipenv:
pipenv install
pipenv shell
3. Apply database migrations:
python manage.py migrate
4. Load sample data (optional):
python manage.py loaddata database_dump.json
5. Run the development server:
python manage.py runserver
6. Open your browser and navigate to:
http://localhost:8000
📋 Project Features
Core Functionality
  • User authentication and profile management
  • Follow/unfollow system for community building
  • Post creation with citations and replies
  • Expertise area classification (AI-powered)
  • Truth rating system for content reliability
  • Community user ratings and feedback
  • Admin moderation and user banning
  • Content timeline and feed

🗄️ Database Models
Key Models
SocialNetworkUsers

Extended user model with follow relationships, community memberships, and moderation controls

Posts

Content entities with author, expertise classification, truth ratings, citations, and replies

ExpertiseAreas

Content categories for AI-powered classification and community organization

TruthRatings

Community-driven reliability ratings (high, medium, low)

UserRatings

User feedback and ratings on posts

🧪 Testing
Run Unit Tests
python manage.py test
Note:
Tests use the fixture "database_dump.json". You can disable failing tests and enable them one by one to debug specific functionality.
📚 Technology Stack
  • Backend: Django (Python web framework)
  • Database: SQLite (default) / PostgreSQL (production)
  • Frontend: Django Templates with HTML/CSS
  • API: Django REST Framework (for programmatic access)
  • Dependencies: Managed with pipenv
